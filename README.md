# CPU Scheduling Simulator

A web-based CPU Scheduling Simulator that visualizes and compares different operating system scheduling algorithms.

## 🚀 Features
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Round Robin
- Priority Scheduling
- Gantt Chart Visualization
- Waiting Time & Turnaround Time Calculation
- Interactive UI

## 🛠 Tech Stack
Frontend:
- React.js
- Vite
- Tailwind CSS
- Recharts (for charts)

## 📊 Algorithms Implemented
- FCFS
- SJF (Non-preemptive)
- Round Robin
- Priority Scheduling

## ⚙️ How to Run Locally

1. Clone the repository:
   git clone https://github.com/YashMehra15/CPU-Scheduling-Simulator.git

2. Navigate to project folder:
   cd CPU-Scheduling-Simulator

3. Install dependencies:
   npm install

4. Run development server:
   npm run dev

5. Open in browser:
   http://localhost:5173

🚀 Live Demo

https://cpu-scheduling-simulator-smoky.vercel.app/

## 🎯 Purpose
This project helps visualize CPU scheduling algorithms and understand performance metrics like average waiting time and turnaround time.
